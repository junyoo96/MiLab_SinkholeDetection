첨부에 시뮬레이션 파일 추가본을 보내드립니다.



저번까지는 homogenous 매질에 안테나 실제제품에 대한 피팅 모델들이 기반으로 된 파일들이었고,

오늘 보내는 파일은 inhomogeous 매질 설정방법, 그리고 전파방사가 실제 제품 모델에 의해서가 아니고, ideal source에 의한 방사를 전제로 한 파일이 되겠습니다.

안테나 제품 피팅모델은, 동작주파수가 정해져 있다는 점 때문에 여러 종류의 데이터를 얻는데 한계가 있습니다.

본 첨부파일과 같이 ideal source가 전제되면, 아무 주파수나 설정이 가능하므로 여러 데이터를 축적하기 위해서는 ideal source 데이터가 중심이 되어야 할 것입니다.

  

gprMax 홈페이지 Docs 메뉴에서 Input file commands로 들어가서

1. ideal source 설정 관련 command (#waveform, #hertzian_dipole, #rx, #src_steps, #rx_steps)

2. inhomogeneous 매질 설정 관련 command (#soil_peplinski, #fractal_box)

3. 전파방사 스냅샷 관련 command (#snapshot)

등의 명령어를 확인하시고, 파라미터를 조금씩 바꿔가면서 Geometry 파일(.vti)을 Paraview에서 확인해보면 금방 파악되리라 생각합니다.

3번의 snapshot 기능은 꼭 필요한 건 아니고, inhomogeneous 매질에서의 전파방사 모습을 확인해 보고 싶을 때 옵션으로 해볼 수 있습니다만,

GPU로 실행할 경우 에러가 발생하니 참고바랍니다.
